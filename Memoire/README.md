# Memoire(备忘录)

#### 项目介绍
Android实现一个简单的备忘录，最低API等级：16(Android 4.1)


#### 使用说明

1. 新建，进入新建备忘录界面（注意：标题和内容有字数限制，且标题不能为空）
2. 点击某条记录，会进入编辑修改页面
3. 长按某条记录，询问：是否删除
